## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm

  ```sh
  npm install npm@latest -g
  ```

- node (project use version 20.17.0) - https://nodejs.org/en/download/

### Installation

1. Clone the repo
   ```sh
   git clone xxx
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

3. Start app in local
   ```sh
   npm run dev
   ```
