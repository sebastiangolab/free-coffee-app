## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

-  npm

   ```sh
   npm install npm@latest -g
   ```

-  node - https://nodejs.org/en/download/

### Installation

1. Clone the repo
   ```sh
   git clone xxx
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

3. Start app in local
   ```sh
   npm run dev
   ```
